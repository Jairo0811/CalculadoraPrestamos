LoanCalc RD

Simulador profesional de préstamos amortizados, reconstruido con Angular moderno a partir de una idea nacida en 2020.

Origen del proyecto

Este proyecto inició como un ejercicio práctico propuesto por Gerson Santos Mateo para fortalecer el aprendizaje de Angular.

“Hacer una calculadora de préstamos amortizada donde el usuario pueda insertar la cantidad que desea prestada, seleccionar los años del préstamo y elegir el tipo: hipotecario, automotriz o personal.”
